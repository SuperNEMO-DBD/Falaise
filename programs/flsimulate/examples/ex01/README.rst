==========================
FLsimulate example 01
==========================

Run the simulation:

.. code:: shell

   $ flsimulate \
      -d "flsimuex01@$(pwd)" \
      -c flsimulate-ex01.conf \
      -o flsimuex01.brio
..

Visualize simulated data:

.. code:: shell

   $ flvisualize -i flsimuex01.brio
..


.. end
