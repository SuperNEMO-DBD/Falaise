==========================================================================
Realistic geometry models for SuperNEMO source foils (strips/pads)
===========================================================================

Two options are available through the Falaise variant system:

* ``flat`` : All LAPP-style and ITEP-style source foils are considered flat.
* ``snrs1`` : All ITEP-stype source foils (but strip #2) are modelled using the
  specific geometry models form the SNRS package. This model still use the *flat* model for all LAPP-style source foils and the ITEP-style strip #2.


.. end
